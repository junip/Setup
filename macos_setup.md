- Install Homebrew
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
# this is a self explanatory script that will install required packages.
# a package manager for the macos
```
- Install Git 
```
brew install git
```
